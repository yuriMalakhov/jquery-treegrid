TreeGrid jQuery plugin

Overview

Что такое TreeGrid? Это плагин jQuery для отрисовки дерева в таблице.

Зачем это нужно? Обычными таблицами не всегда удобно пользоваться, для отображения информации в которых есть родительские и дочерниии эллементы. А обычные деревья, не всегда удобны для отображения больших данных.

Возможности

  • Создания девева из HTML

Usage


        <script type="text/javascript" src="js/jquery.js"></script>
        <script type="text/javascript" src="js/jquery.treegrid.js"></script>
        <script type="text/javascript">
          $('.tree').treegrid();
        </script>
Как должна выглядеть таблица.

Configuration Settings

Settings

ПараметрТипЗначение по умолчаниюОписание
initialState String expanded Изначальное состояние таблицы
'expanded' - Все ветки раскрытые
'collapsed' - Все ветки свернутые
expanderTemplate String <span class="treegrid-expander"></span> Елемент при нажатии на который будет проискодить раскрытие/свертывание веток
indentTemplate String <span class="treegrid-indent"></span> Елемент который будет ставиться как отступы в зависимости от глубины вложения nodы

Events

Public methods

Название Описание Пример
getRootNodes Возвращает корневые ветки 
            $('.tree').treegrid('getRootNodes').treegrid('expand');
getNodeId Возвращает id ветки 
            if($('#node-2').treegrid('getNodeId')===2){
              //DO SOMETHING
            };
getParentNodeId Возвращает id родительской ветки или null если ветка корневая 
            if($('#node-2').treegrid('getParentNodeId')===null){
              //DO SOMETHING
            };
getParentNode Возвращает родительскую ветку или null если ветка корневая 
            $('#node-2').treegrid('getParentNode').treegrid('collapse');
getChildNodes Возвращает child nodes или null если ветка листик 
            $('#node-2').treegrid('getChildNodes').treegrid('expand');
getDepth Возвращает глубину вложености ветки 
            //открыть все до 2ой вложености
            $('.tree').find('tr').each(function(){
              if ($(this).treegrid('getDepth')<2){
                $(this).treegrid('expand');
              }
            });
isLeaf Является ли ветка листиком 
            //сапятать все файлы
            $('.tree').find('tr').each(function(){
              if ($(this).treegrid('isLeaf')){
                $(this).hide();
              }
            });
isLast Является ли ветка последним элементом среди всех элементов с этим родителем 
            //сапятать все последние елементы
            $('.tree').find('tr').each(function(){
              if ($(this).treegrid('isLast')){
                $(this).hide();
              }
            });
isExpanded Раскрыта ли ветка 
            if($('#node-2').treegrid('isExpanded')){
              //DO SOMETHING
            };
isCollapsed Закрыта ли ветка 
            if($('#node-2').treegrid('isCollapsed')){
              //DO SOMETHING
            };
isOneOfParentCollapsed Закрыта ли хоть одна из родительских веток 
            if($('#node-2').treegrid('isOneOfParentCollapsed')){
              //DO SOMETHING
            };
expand Раскрыть ветку 
            $('#node-2').treegrid('expand');
collapse Закрыть ветку 
            $('#node-2').treegrid('collapse');
toggle Закрыть ветку если она открыта и открыть, если закрыта 
            $('#node-2').treegrid('toggle');
render Перерисовать ветку и всех ее детей 
            $('#node-2').treegrid('render');
